What data type is collected?

Chatty collect participants information like:

  • chat title
  • participants names
  • messages number sendt by participants
  • characters number used in chat by participants
  • words number used in chat by participants
  • gifs number used in chat by participants
  • photos number posted in chat by participants
  • links number posted in chat by participants
  • 3 most common words used in chat by participants divided by length of the word
  • most common reaction used in chat by participants

How the data is collected?

user is uploading his own chat file downloaded from his facebook profile. Chatty analyzing numbers and types of information and provide nice way to display extracted data

How the data is stored?

Chatty is free project where no money is made. Data is stored on a database server and each 15 minutes the database is cleared from not deleted files. User on the end of the proccess has ability and it's recommended to delete/clean database

Whether data is shared with any other parties?

Your data is not shared with any other parties. Chatty is not even saving a file you uploaded. Data extracted from file has no sense except statistical information.
